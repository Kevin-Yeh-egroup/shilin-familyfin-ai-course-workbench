# 文山職能工作坊好理家在 AI 課程講綱草稿

這個 repo 放置檢視用的靜態課程頁，用於討論「只用好理家在」可如何協助文山職能工作坊工作人員進行 AI 協作、個案歷程紀要、時間資源盤點、服務對象互動與資源連結課程。

本版依夥伴回饋拆成兩份內容：第一份為邀課單位評估課程安排使用的講綱草稿，第二份為內部備課與教學設計討論用的設計說明。

## 內容

- `index.html`：講綱草稿頁，保留邀課單位需要的課程目的、適用對象、效益、兩天安排與好理家在可提供的協助。
- `design-notes.html`：設計說明頁，放工具安排邏輯、公開資料查核、示範流程與後續需確認資料。
- `wenshan-course-outline-v0.2.md`：兩堂課、每堂約 3 小時的講綱草稿。
- `wenshan-familyfin-two-day-course-outline-v0.2.docx`：頁面右上角下載用的 Word 講綱。
- `robots.txt` 與 `vercel.json`：公開可連結，但保留 review-stage noindex 設定。

## 邊界

- 不包含真實個案資料。
- 不包含登入資訊、測試紀錄或內部系統資料。
- 工具輸出僅作課程討論與人工審核草稿，不作心理診斷、法律結論或醫療用藥建議。
