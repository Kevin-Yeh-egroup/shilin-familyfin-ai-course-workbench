# 士林地檢署心理師好理家在 AI 課程工作台

這個 repo 放置第一版公開檢視用的靜態工作台，用於討論「只用好理家在」可如何協助士林地檢署心理師夥伴進行 AI 與資訊化課程。

## 內容

- `index.html`：工作台教學引導頁。
- `course-outline-v0.1.md`：兩堂課、每堂約 3 小時的第一版課綱草稿。
- `robots.txt` 與 `vercel.json`：公開可連結，但保留 review-stage noindex 設定。

## 邊界

- 不包含真實個案資料。
- 不包含登入資訊、測試紀錄或內部系統資料。
- 工具輸出僅作課程討論與人工審核草稿，不作診斷、法律結論或醫療建議。
